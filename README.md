# Phat-ReactRedux-Boilerplate
My own boilerplate for react redux

## Guide:
```
git clone https://github.com/iPhatty/Phat-ReactRedux-Boilerplate.git

cd Phat-ReactRedux-Boilerplate

npm start
```

## Create production app using:
```
npm run build
```
